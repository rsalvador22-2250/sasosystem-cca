JSON File Format Guide

File Format Requirements

Your JSON file must be an array of objects with the following fields:

Field NameRequiredDescriptionExample
name✅ YesFull name of student"Juan Dela Cruz"
student_number✅ YesStudent ID number"2024-00123"
course✅ YesCourse/Program"BS Information Technology"
offense✅ YesType of offense"Tardiness"
date❌ OptionalDate registered"January - 15 - 2024"

Date Format Options

// Recommended Format (Best for filtering)
"date": "January - 15 - 2024"
"date": "February - 28 - 2024"

// Standard Date Formats
"date": "2024-01-15"
"date": "2024/02/28"
"date": "Jan 15, 2024"

// Leave empty (uses current date)
"date": ""

How to Upload

Step 1: Click the "JSON Upload" button (blue button)

Step 2: Select your .json file from your computer

Step 3: Wait for confirmation message

Step 4: Records will appear automatically in the table

Important Notes

⚠️ File must end with .json extension

⚠️ Must be a valid JSON array (starts with [ and ends with ])

⚠️ Field names are case-sensitive

⚠️ Email field will be set to "N.A" automatically

⚠️ Test with 1-2 records first before bulk uploading
